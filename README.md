## Software and Projects

### ViCaRe — VISSIM Calibrator and Reporter

ViCaRe is a VISSIM calibration and reporting automation software designed to help traffic engineers automate repetitive simulation workflows while maintaining full engineering control.

Key capabilities include:

- Automated VISSIM calibration
- Genetic Algorithm and Hall-of-Fame calibration approaches
- Multi-scenario calibration
- Validation using up to 4 Measures of Effectiveness
- Penalized error evaluation
- Dynamic mutation / increment control
- Excel-based reporting
- Transparent, engineering-focused workflow
