This paper presents a long-term memory approach for calibrating driving behavior parameters in VISSIM models. ## Software and Projects

### ViCaRe — VISSIM Calibrator and Reporter

**ViCaRe** is a VISSIM calibration and reporting automation software designed to help traffic engineers automate repetitive simulation workflows while maintaining full engineering control.

ViCaRe supports:

* Automated VISSIM calibration
* Genetic Algorithm calibration
* Long-term memory / Hall-of-Fame calibration approaches
* Multi-scenario calibration
* Validation using up to 4 Measures of Effectiveness
* Penalized error evaluation
* Dynamic mutation / increment control
* VISSIM-COM automation
* Excel-based results reporting
* Transparent, engineering-focused workflows

No AI black boxes.
Just engineering logic, math, automation, and transparent reporting.
